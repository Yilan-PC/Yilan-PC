品名：i Master key 鑰匙圈 usb / type-c 雙頭 體積大約 1.5 公分長

架構：不需什麼樣的晶片或怎麼樣的物理支持，將所有必須使用到的套件都已設計在 usb 內，並可獨立運行於 android / windows / linux / mac os x / ios / chrome os / 
for arm and x86 上. 

挑戰與任務。
這是我的概念雛型商品，至於要把概念轉換為實體商品，則需要一連串的手法與技術配合才能實踐。
並非我們不使用或買不起實體金鑰，而是看見了更多大眾的需求，至少提供一個可用度高的安全性解決方案。也幫助迷糊的人，不會失去帳號的訪問權限。鑰匙圈的設計，它就是一個吊飾。也是你的密碼庫。

商品概念。類似硬體金鑰的替代品。卻是普遍網站的必需品。實用度不亞於硬體金鑰。但安全性略遜一籌。此商品是繼承密碼usb概念而演伸出的分支。在硬體的使用上，將它發揮出最大價值。

製作自動輸入密碼的 usb 安全性不能與其他高單價商品相比但實用度對於較重視效率的人來說則是一大福音。要能夠自動分辨網站，app，或者資料欄位如：地址欄等等，不須使用外力幫助，獨立運行。
不用時時監控畫面，而是當需要時再接入usb or type c 界面接口。
優點：價格低廉，高效率，方便，變相也提高安全性。
在關於隱私個資的欄位快速輸入已經準備好的資料。附加功能：產生匿名資料，以及測試用visa card 也可產生合乎規定的英文地址，及郵遞區號快速完成註冊國外網站之須求。
不須網站支援硬體金鑰，即可使用強密碼。快速輸入，且保有資料控制權。這些輸入值，我們都能夠獲取。


品名：DeepNotes 深度筆記

![23091_s_DSC_6126](https://hackmd.io/_uploads/BkiJbKJfye.jpg)


概念：此商品為原實體筆記的更名，改為深度筆記，是一種客製化，我設計的。內頁，擁有各式網路應用
簡易的qr 掃描即可使用內建的工具。這本深度筆記，便於攜帶，也用了雙向存儲概念，例如：掃描 qr 後會出現一個錄音框框，你即可把要錄製的音效點擊錄音按鈕，錄製起來。所以這不是傳統概念的筆記本，它更像是半虛擬化的一本工具書。當然錄音的框框為什麼會有，這就要我們尋找網路上的線上應用，來達成這個功能，我們可以簡易做些網站或者app整合這些功能。

應用場景 2 ：在危險的 wifi 下，卻又要發送大檔案給某人，而它跟我在不同的區域網路內。發送超過4 GB 以上的文件，最好是能夠送入他的 inbox 此時，相對應的QR Code 簡單一掃 即刻出現對應之應用（網頁板）免安裝，讓你天崖咫尺，無國界的寄送大檔，有些MAIL提供商也是有提供夾帶大檔的功能。但其實換句話說，使用雲端硬碟在分享即可，直接存入對方雲端硬碟，如同百度的作法一樣。整個中 國就是我的區域網路 LAN 。但在台灣，此功能必須整合入我的 DeepNotes 。
